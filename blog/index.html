How I Dive Into Python World

Aug 27th, 2016 5:24 pm

Apart from some technical stuff ,now i want to share something else that is “How i dive into python world”. Actually before entering to college,i know python is something that we hear from our childhood.I was only stick with “C” and “C++”. I heard about “PYTHON” from one of my friend. Then in curiosity,I started learning it and soon i find it is more easier and flexible than other programming languages.The most important feature of python which attracts me most is it is open-source.Soon i fell love with it and i started writing programs with it and i love to do that.I started learning it from various sources like python docs which is available in python.org website,video tutorial from the “NEW BOSTON”,and books like think python,byte of python and many more.. Web-development is also easy with it by it’s “DJANGO” framework and “pygame” helps users to creates interactive games. There are many python user group almost in every countries and cities around the world . They also organize some event almost in every year in which they discuss about various stuffs,talks and organize some workshop based upon python.India also organize “PYCOON INDIA”in which the python users join and share their knowledge and i think it is the beauty of to be a part of the open-source family . Afterall,I feel lucky to be a part of this family.

A Funny App

Aug 24th, 2016 9:49 pm

Hello folks,

After making of two games ,I have completely fallen in love with canvas and javascript .So i had decided to make another game by using this.This time i thought to make a funny simple game. So i have decided to make a game name as “KNOW YOUR STRENGTH” .Actually In this game a button is attached with a meter containing a ball inside it. You have to hit that button with a hammer .As u hit that button ,the ball inside the meter grows up.The meter is divided into 4 part named as “WEAK”,”AVERAGE”,”STRONG”,”SUPER STRONG”.The ball inside the meter stops in a definite part and according to that part the meter gives your strength. So simple game.
